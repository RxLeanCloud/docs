# RxLeanCloud Docs

### 在本地运行

首先请安装 [Ruby](https://www.ruby-lang.org/en/documentation/installation/), [Bundler](http://bundler.io/), and [npm](https://www.npmjs.com/get-npm).

Clone 当前 Repo， 然后安装 Jekyll:

```
bundle install
npm install
```

然后再运行一下：

Linux/OS X:
```
npm start
```

Windows：
```
npm run dev-win
```

最后在浏览器打开 http://localhost:4000/ 即可进行实时预览。
